# Buddy Assets

The `mouse-*.png`, `fox-*.png`, and `frog-*.png` frames are original local pixel frames created for Lensys.

License reference verified while implementing: [Critters - Mouse by Foozle](https://foozlecc.itch.io/realm-of-critters-mouse-warrior), distributed as Creative Commons Zero (CC0). The local frames are intentionally small and simple so the app has bundled buddy artwork without requiring an external itch.io download flow.
